---
title: 和现有码表校对
sidebar: { order: 8 }
---

有时我们会希望使用自动拆分系统复原一个现有的输入方案，且我们已经有了这个方案的码表。此时我们可以利用这个现成的码表来校对，确保我们复原的是正确的。

切换到「校对」页面，点击「导入码表」，系统会比对当前方案所产生的编码结果和码表是否相同，并把不同的显示出来以便调整。
![](https://images.tansongchen.com/1769552198.png)

在一开始校对的时候，可能因为错误数量较多而感到无从下手。此时可以将「校对范围」切换到「成字部件」完成校对之后，再切换到「所有汉字」。这是因为对于绝大多数方案来说部件的拆分决定了复合体的拆分，因此先保证成字部件（大约几百字）编码正确之后基本上其他汉字也会正确。
