# 字形数据的自定义

在汉字自动拆分系统中，对于一个汉字，系统可能会提供它的一个或多个字形。一般来说，每个汉字只有一个字形，例如「十」就只有一个字形，它是一个「基本部件」：

![](https://images.tansongchen.com/1769395172.png)

但是，也有可能有多个字形。例如，「少」有一个基本部件的字形，也有一个复合体的字形（看成是上下结构）。

![](https://images.tansongchen.com/1769395297.png)

一个字究竟采取哪个字形，在不同的方案之间有一定的灵活空间。例如，一个采取分部取码的方案通常会自己定义哪些字可分、哪些字不可分。因此，方案可以在系统提供的多个内置字形中任意选择一个，也可以自己从头创建一个。在没有自定义的情况下，默认选取系统字形中的第一个。自定义的菜单如图所示：

![](https://images.tansongchen.com/1769395649.png)

## 选择系统字形中的一个

前面的几个选项表明可以选择系统中现有的若干个字形中的一个。选择后，该字形会出现在「自定义字形」列中。

## 自己创建一个

后面的几个选项表明可以自己从头创建一个字形。其含义为：

- 自定义衍生部件：以某个已有的部件为蓝本，增加和减少少量的笔画形成新的部件；
- 自定义拼接部件：把某些部件按一定的空间位置关系拼接起来，但结果仍然看作是一个部件；
- 自定义复合体：把某些部件或复合体按一定的空间位置关系拼接起来，成为一个复合体；

## 新建字符

有的情况下我们想要使用的一个字根是系统的数据库中没有的，这时候我们就需要新建字符。

![](https://images.tansongchen.com/1769396048.png)

新建条目需要先为该条目提供一个名称以及定义类型。

### 部件

如果定义的是部件类型，则需要定义它所包含的各个笔画。一般来说，我们只需要选择某个部件作为源字，然后引用它的笔画即可，字形会在左方的预览区实时预览。

### 复合体

如果定义的是复合体类型，则需要定义它的分部方式。复合体类型的分部方式允许定义多个。如果不对复合的笔顺进行定义，则默认为第1部笔顺+第2部笔顺（+第3部笔顺）。
