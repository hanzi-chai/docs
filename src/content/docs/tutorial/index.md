汉字自动拆分系统（<https://chaifen.app>）是一个网页应用程序，您可以用它来设计和优化汉字输入方案。首次打开网站时，您将看到中间的 Logo 和左侧的空方案列表。

![](https://images.tansongchen.com/1769386624.png)

# 新建方案

有三种方式来添加方案：

- 点击「新建」将弹出一个对话框引导您从一些模板快速创建方案
- 点击「示例」则可以复制一份示例方案，并基于它制作方案
- 点击「导入」可以导入别人制作的或者自己先前从本系统中导出的 YAML 格式的方案文件

添加成功后，方案列表中会出现一个新方案。点击任一方案即可进入编辑器界面，点击右侧红色删除图标则将移除该文件。不同方案之间数据是相互独立的。

这里我们点击「示例 - 形音码 - 米十五笔」，复制一份米十五笔的示例方案，并点击这个方案进入编辑器页面，以便后续的演示。

# 基本信息的编辑

在首页可以编辑方案的基本信息，包括方案名称、作者、版本、描述等等。
![](https://images.tansongchen.com/1769552494.png)

# 基本操作

在每一页的右上角，都有关于方案文件的操作工具栏：
![](https://images.tansongchen.com/1769552555.png)

1. 「校验」：检查当前方案配置文件格式是否正确。如果手动编辑了 YAML 格式的文件后导入，或从较旧的版本升级，「校验」通常能提供有用的调试信息。
2. 「环行」：将方案发送到系统的核心算法库再发回来，看结果是否等价。这主要是用于开发调试，普通用户可以不用管。
3. 「替换」：导入一个 YAML 文件替换当前方案，当前方案被覆盖。如果导出后手动编辑了 YAML 文件，可以使用此选项。
4. 「导出」：将方案文件导出用于分享或结合命令行计算工具使用。
5. 「重置」：如果方案是从示例文件复制来的，点击此可以一键将当前方案重置为初始状态。
6. 「撤销」：撤销对方案做的上一次修改。注意，方案的历史记录只会保存 10 项，更多的将不会被保存，且在页面刷新之后历史记录就会丢失。
7. 「重做」：恢复对方案做的上一次修改。
